# Archero Project Initialization

Project skeleton and art import prepared.

Open this folder with Unity 2022.3.x. The editor script in `Assets/Editor/ArcheroProjectInitializer.cs` runs once and creates the initial prefab library plus `Assets/Scenes/Bootstrap.unity`.

This state is initialization-only:

- Imported art resources under `Assets/Art`
- Empty project folders for future development
- Initial prefab generation from imported models
- A bootstrap sample scene generated from imported assets
