# Style Guide

## 一句话方向

做一个明亮、干净、低模奇幻地牢射击游戏。画面不追求真实，重点是俯视角下的角色轮廓、弹幕辨识和技能反馈。

## 视觉原则

- 低模几何体，少贴图细节，多靠颜色块区分对象。
- 角色和怪物要有清楚轮廓，俯视角下也能一眼区分大小、威胁和阵营。
- 地牢环境保持低饱和，玩家、怪物、投射物和掉落物保持更高对比度。
- 玩家攻击特效用青色、金色、白色为主；敌方危险弹幕用红色、紫色、橙色为主。
- UI 使用干净的扁平按钮和图标，不要做成写实皮革/金属厚重风格。

## 俯视 3D 约束

- 玩家和怪物模型都必须是 3D 模型，不使用 2D 精灵冒充主单位。
- 摄像机建议 45-60 度俯视，角色头部和武器要能被看见。
- 场景障碍高度不能遮住玩家，必要时做透明/压低处理。
- 子弹和技能特效要比角色更亮、更简单，避免和地面纹理混在一起。

## 推荐资源搭配

- 玩家：Quaternius modular character Adventurer。
- 武器：Quaternius Medieval Weapons 的木弓、金弓、邪恶弓。
- 普通怪：Blob/Mushnub/Orc/Cactoro/Wizard/Ghost。
- Boss：MushroomKing/Dragon/Demon。
- 地牢：Quaternius Modular Dungeons。
- UI：Kenney UI Pack + Kenney Game Icons。
- 粒子：Kenney Particle Pack，用作命中、升级、元素箭、Boss 弹幕。

## 资源拒绝标准

- 写实 PBR、复杂法线、照片纹理。
- 和 Quaternius 低模比例差异过大的模型。
- 直接使用 Archero 原版图标、UI、角色、装备图。
- 过暗、过脏、细节太碎，俯视角看不清的资产。
- 风格强烈的免费素材，例如厚重暗黑、赛博朋克、二次元卡通，不放入正式目录。

## 两周 Demo 的美术完成度目标

- 一个完整地牢房间视觉基准。
- 玩家携带弓，能跑动、停下射击。
- 6-8 个普通怪，1 个精英怪，2-3 个 Boss。
- 箭矢、命中、升级、死亡、拾取都有基础反馈。
- 技能选择界面有图标、按钮和明确稀有度/类型色彩。
