# Archero-like Art Resource Library

这个目录用于两周版 3D 俯视角弓箭传说-like Demo 的美术资源归档。

## 风格定位

- 主风格：低多边形、明亮奇幻、俯视角可读性优先。
- 主资源体系：Quaternius CC0 低模 3D 资源。
- UI/VFX 补充：Kenney CC0 UI、图标、移动摇杆与粒子贴图。
- 禁止混入：写实高模、暗黑恐怖、二次元描线、直接仿 Archero 的角色/UI/图标。

## 目录职责

- `00_SourcePackages`: 原始下载包、授权文件、资源清单。
- `01_StyleGuide`: 风格规范和后续美术约束。
- `02_Characters/Player`: 玩家 3D 模型、玩家动画。
- `03_Enemies`: 普通怪、精英怪、怪物贴图。
- `04_Bosses`: Boss 3D 模型。
- `05_Weapons_Projectiles`: 弓、箭、投射物。
- `06_Environment`: 模块化地牢、障碍、宝箱、陷阱。
- `07_VFX`: 粒子贴图、材质参考、后续 Unity/Unreal 粒子 prefab。
- `08_UI`: 技能图标、按钮、移动摇杆、基础 UI。
- `09_Reference`: 官方预览图和风格参考。
- `10_Working`: 待审资源、风格不匹配资源。
