# Virtual Joystick Art

Curated UI art for the Archero-like one-finger virtual joystick.

## Recommended set

- `Selected/VJ_Pad_Primary.png`
- `Selected/VJ_Nub_Primary.png`

This primary set uses the Kenney Mobile Controls Style H circle joystick. It has enough contrast for a top-down 3D room while staying quiet enough for a mobile HUD.

## Variants

- `Variants/Light_Minimal`: lower visual weight, useful if the combat floor is dark.
- `Variants/Dark`: stronger contrast, useful over bright floors.
- `Variants/Mixed_Contrast`: light pad with dark nub for readability testing.
- `Variants/Highlight`: yellow and purple ring versions for pressed/active feedback.
